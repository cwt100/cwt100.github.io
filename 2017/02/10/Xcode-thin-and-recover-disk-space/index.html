Xcode thin and recover disk space

Xcode

Target

Macbook Pro 空間有限，再加上使用 Xcode 開發 iOS ，
很容易造成硬碟空間不足，導致系統變慢。

本篇主要目的是將 Xcode 瘦身，釋放硬碟空間。

Action Items

  1. Delete 不常用的 iOS 版本
  2. Delete 不需要的 Archives 文件
  3. Error Handle - 容量沒有增加

Quick Start

Delete 不常用的 iOS 版本

這裡放的是連接iOS Device 所產生的檔案，不常用的版本可以刪除，再次連接設備會再產生。
路徑 ->
~/Library/Developer/Xcode/iOS DeviceSupport

Delete 不需要的 Archives 文件

這裡放的是 APP 打包生成的文件，可以刪除不需要要的打包文件。
路徑 ->
~/Library/Developer/Xcode/Archives

Error Handle - 容量沒有增加

刪除上面列出的檔案後，至少可以清出好幾“G”的空間。
但是實際去看 MacBook Pro 的儲存空間，卻沒有任何改變？？？

解決方式

使用磁碟工具程式 -> 選擇要修理的磁碟 -> 修理工具
重新開機後，就可以在儲存空間中，發現可用空間增加了。

  1. 工具程式

  2. 磁碟工具程式

  3. 修理工具

Postscript

使用此方式，我這次就清掉了 20G 以上的空間，
原因都是累積太多舊版本的 iOS 實機檔案，
大家有空可以去看看 iOS DeviceSupport 是不是有太多舊版本的檔案，在占用你的 Mac 空間。
